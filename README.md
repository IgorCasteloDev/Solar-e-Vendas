# 🌞 Solar e Vendas

Landing page moderna desenvolvida para apresentar serviços de **energia solar**, com foco em **captação de clientes e conversão de leads**.

## 🚀 Demonstração

🔗 Acesse o projeto online:
https://solarevendas.netlify.app

## 📌 Sobre o projeto

O **Solar e Vendas** é um site institucional criado para empresas do setor fotovoltaico.
O projeto tem como objetivo **transmitir confiança, destacar benefícios da energia solar e incentivar o contato direto com o cliente**.

## ✨ Funcionalidades

* 📱 Layout totalmente responsivo
* ⚡ Interface rápida e otimizada
* 💬 Botões de contato direto (ex: WhatsApp)
* 🎯 Estrutura pensada para conversão
* 🌿 Seções explicativas sobre energia solar

## 🖥️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
